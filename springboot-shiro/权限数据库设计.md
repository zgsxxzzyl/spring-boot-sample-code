> 用户组 => 用户 => 角色 => 权限
## `t_sys_user `
| 字段名   | 类型    | 说明         |
| -------- | ------- | ------------ |
| id       | int     |              |
| name     | varchar | 用户名       |
| password | varchar | 密码         |
| salt     | varchar | 盐           |
| locked   | tinyint | 账户是否锁定 |

## t_sys_permission
| 字段名      | 类型    | 说明       |
| ----------- | ------- | ---------- |
| id          | int     |            |
| name        | varchar |            |
| permission  | varchar | 权限字符串 |
| available   | tinyint | 是否可用   |
| description | varchar | 描述       |

| 类型 | 字符串 |
| ---- | ------ |
| 增   | 0001   |
| 删   | 0010   |
| 改   | 0100   |
| 查   | 1000   |
